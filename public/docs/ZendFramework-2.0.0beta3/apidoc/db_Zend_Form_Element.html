addPrefixPath( string $prefix, string $path, string $type = null ) : \Zend\Form\Element

Add prefix path for plugin loader

If no $type specified, assumes it is a base path for both filters and validators, and sets each according to the following rules: - decorators: $prefix = $prefix . '\Decorator' - filters: $prefix = $prefix . '\Filter' - validators: $prefix = $prefix . '\Validator'

Otherwise, the path prefix is set on the appropriate plugin loader.
