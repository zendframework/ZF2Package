\Zend\Server\Reflection\AbstractFunction
jump to top

Package: Zend\Server\Zend\Server\Reflection

Function/Method Reflection

Decorates a ReflectionFunction. Allows setting and retrieving an alternate 'service' name (i.e., the name to be used when calling via a service), setting and retrieving the description (originally set using the docblock contents), retrieving the callback and callback type, retrieving additional method invocation arguments, and retrieving the method {@link \Zend\Server\Reflection\Prototype prototypes}.
